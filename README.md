#🎓 AI Career Mentor Chatbot
## About the Project

This project is a beginner-friendly AI-powered Career Mentor Chatbot built using Python and Streamlit.

It is designed to help students understand career paths in Artificial Intelligence, Data Science, Machine Learning, coding skills, internships, and project building.

The chatbot uses Natural Language Processing (NLP) techniques like TF-IDF Vectorization and Cosine Similarity to match user queries with the most relevant answers from a prepared FAQ dataset.

Through this project, I gained hands-on experience in building an intelligent system that can understand user input and respond meaningfully.

## Project Workflow
User enters a question
Text is cleaned and preprocessed
TF-IDF converts text into numerical vectors
Cosine Similarity finds the closest matching FAQ
Chatbot returns the best possible answer
Confidence score is displayed for transparency
