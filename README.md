## 🎓 AI Career Mentor FAQ  Chatbot 
About This Project

As a first-year B.Tech student in AI & Data Science, I wanted to build a beginner-friendly AI project that helps students like me understand the path toward a career in Artificial Intelligence and Data Science.

This project is an AI-powered Career Mentor Chatbot developed using Python and Streamlit. It answers common questions related to AI, Machine Learning, Data Science, coding skills, internships, and project building.

The chatbot works by using Natural Language Processing (NLP) techniques like TF-IDF Vectorization and Cosine Similarity to match user questions with the most relevant answers from a prepared FAQ dataset.

Through this project, I gained practical experience in building an intelligent system that can understand user queries and respond in a meaningful way.

## Project Workflow
 User enters a question
 The text is cleaned and preprocessed
 TF-IDF converts text into numerical vectors
 Cosine Similarity finds the closest matching FAQ
 The chatbot returns the best possible answer
 Confidence score is displayed for transparency
